Define Web Browser, Web Site, Web Page, Web Server. =A Web Browser is a software program that helps the user to find stuffs like pictures, videos, articles etc. For Example Google Chrome, Firefox, Opera. =A Web Page is an online page that can be one or more pages according to the user in a browser. =A Web Site is similar to Web Page that also contain one or more sites in a browser according to the User. =A Web Server is use for providing information and also for creating a new website.
